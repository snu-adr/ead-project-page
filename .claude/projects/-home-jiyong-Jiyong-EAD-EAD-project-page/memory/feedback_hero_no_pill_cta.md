Hero 섹션에 highlight pill 뱃지와 CTA 버튼을 넣지 않는다.

**Why:** 사용자가 명시적으로 금지. content.json에 highlights 데이터가 있더라도 Hero에 렌더링하지 않음.

**How to apply:** Hero 컴포넌트 작업 시 pill, badge, CTA 버튼 등의 인터랙티브/뱃지 요소를 추가하지 않는다.
