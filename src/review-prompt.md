# EAD 사이트 디자인 리뷰 프롬프트

## 프로젝트 컨텍스트

EAD(Evolutionary Autonomous Driving)는 서울대학교 ADR 연구실의 자율주행 프로젝트 소개 사이트입니다.

**타겟 오디언스**: 국내 자율주행 업계 종사자, 잠재적 파트너사, 정부/투자 관계자

**전달하려는 인상**: 기술력에 대한 신뢰감 + 협업 개방성

**디자인 톤**: 다크 테마, 자율주행 센서 UI 느낌, 시안(#38bdf8) 액센트

## 리뷰 관점

### Quick 모드
아래 항목만 평가:
1. **정보 위계**: 가장 중요한 메시지가 먼저 눈에 들어오는가?
2. **구조와 레이아웃**: 섹션 구분이 명확한가? 시각적 흐름이 자연스러운가?
3. **전문성 인상**: 첫인상에서 신뢰감과 기술력이 느껴지는가?
4. **명백한 버그**: 텍스트 잘림, 요소 겹침, 빈 공간 낭비 등

### Detailed 모드 (Quick + 추가)
5. **타이포그래피**: 폰트 크기, 굵기, 행간이 적절한가?
6. **색상 대비**: 텍스트가 배경 위에서 충분히 읽히는가?
7. **간격과 여백**: 요소 간 간격이 일관적인가?
8. **세부 디자인**: 배지, 아이콘, 경계선 등이 전체 톤과 어울리는가?


## 제약 조건 (리뷰 시 고려)

- CTA 버튼, Highlight pill은 의도적으로 없는 것 (금지 요소)
- 동영상 플레이스홀더("데모 영상 준비 중")는 실제 영상 전 임시 상태
- 모식도 플레이스홀더("모식도 준비 중")도 임시 상태
- 이메일/GitHub URL은 placeholder (확정 전)
- 로고는 SVG 텍스트 플레이스홀더 (실제 로고 준비 전)
